## üì∏ How to Capture Screenshots

### Method 1: Physical Device (Recommended)

1. Install app on iPhone with required screen size
2. Set up each screen exactly as described
3. Press Volume Up + Side Button simultaneously
4. Screenshots saved to Photos app
5. AirDrop to Mac or download from iCloud

### Method 2: Simulator

1. Open Xcode
2. Choose simulator with correct size:
   - iPhone 15 Pro Max (6.7")
   - iPhone 11 Pro Max (6.5")
3. Run app (Cmd+R)
4. Navigate to desired screen
5. Cmd+S to save screenshot
6. Screenshots saved to Desktop

### Method 3: Xcode Debugger

1. Run app on physical device
2. Navigate to desired screen
3. In Xcode: Debug ‚Üí View Debugging ‚Üí Capture View Hierarchy
4. Export as image
5. Crop to device size

## üì§ Exporting for App Store

### File Format

- **Format:** PNG or JPEG
- **Color Space:** RGB (not CMYK)
- **Compression:** High quality (90%+)
- **Transparency:** None (solid backgrounds)

### Naming Convention

```
1_Main_Screen_6.7inch.png
2_Photo_Detection_6.7inch.png
3_Live_Detection_6.7inch.png
4_Charts_6.7inch.png
5_Settings_6.7inch.png
```

### File Size

- Keep each file under 10 MB
- Typically 500KB - 2MB per screenshot
- PNG recommended for sharp UI

## üí° Pro Tips

### Lighting & Clarity

- Use bright, natural lighting
- Avoid glare and reflections
- Ensure text is crisp
- Test readability at thumbnail size

### Content

- Show real data, not placeholders

### Consistency

- Use same theme (light/dark) across all shots
- Maintain consistent style
- Similar compositions
- Coherent storytelling

### Testing

- View at actual iPhone size (small!)
- Check readability
- Verify no typos
- Get second opinion
