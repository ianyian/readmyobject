Get started with Object Detector Pro Max in 3 simple steps:

üì± Main Features

1. Photo Detection

How to use:

  1. Tap "Take Photo" on the main screen
  2. Take a new photo or select from your library
  3. The app will analyze and highlight detected objects
  4. View count and confidence level for each detection
üí° Tip: For best poker chip counting results, ensure good lighting and place chips on a contrasting background.

2. Live Detection

How to use:

  1. Tap "Live Detection" on the main screen
  2. Point your camera at objects you want to detect
  3. Tap the screen to pause/resume detection
  4. View real-time charts showing detection history and CPU usage
üí° Tip: Tap the camera feed to pause detection while maintaining CPU monitoring. This is useful for reviewing results.

3. Detection Modes

Choose from 10 specialized detection modes in Settings:

  • All Objects: Detects 80+ common objects (people, cars, animals, etc.)
  • Poker Chips: Specialized mode for counting poker chips by color
  • Person: Focus on human detection only
  • Car: Detect vehicles
  • Dog / Cat: Pet detection
  • Bicycle / Motorcycle: Two-wheeled vehicle detection
  • Bird / Horse: Animal detection

4. Settings

Available settings:

  • Detection Mode: Choose what to detect (10 options)
  • Minimum Confidence: Adjust detection sensitivity (10-95%)
    • Higher = Stricter (fewer false positives)
    • Lower = Lenient (more detections, may include errors)
    • Default: 95% (recommended for accurate chip counting)
  • Appearance: Light, Dark, or System (follows iOS settings)

üìä Understanding the Charts

Detection History (Stacked Bar Chart)

Shows the count of detected objects over the last 10 detection cycles:

  • Each bar represents one detection cycle
  • Different colors represent different object types
  • Height indicates count
  • Updates every 0.5 seconds during active detection

CPU Usage (Line Chart)

Monitors real-time CPU usage:

  • Green line shows current CPU percentage
  • Y-axis automatically scales based on usage
  • Continues running even when detection is paused
  • Helps monitor device performance

‚ùì Frequently Asked Questions (FAQ)

Q: Why isn't the app detecting my poker chips?
A: Check these common issues:
  • Make sure you're using "Poker Chips" detection mode
  • Improve lighting conditions
  • Try lowering confidence threshold to 80-85%
  • Ensure chips are on a contrasting background
  • The model is trained on red, blue, green, and black chips
Q: Does the app work offline?
A: Yes! 100% of processing happens on your device. No internet connection required.
Q: Does the app collect my photos?
A: No. All processing happens locally on your device. Your photos never leave your iPhone. See our Privacy Policy for details.
Q: How do I pause live detection?
A: Simply tap anywhere on the camera feed. Tap again to resume. CPU monitoring continues even when paused.
Q: Why is detection slower on my device?
A: Detection speed depends on your device's processor. Older devices may process frames more slowly. The app is optimized for iOS 15.0+, but newer devices (iPhone 12+) will perform best.
Q: Can I detect custom objects?
A: Currently, the app detects:
  • 4 poker chip colors (custom model)
  • 80 common objects from COCO dataset (general model)
Custom object detection is not currently supported but may be added in future versions.
Q: What's the difference between confidence levels?
A:
  • 10-50% (Lenient): Many detections, higher chance of false positives
  • 60-80% (Balanced): Good for general object detection
  • 85-95% (Strict): Fewer detections, but more accurate (recommended for chips)
Q: Why does CPU usage vary?
A: CPU usage depends on:
  • Number of objects in frame
  • Complexity of the scene
  • Your device's current workload
  • Whether detection is active or paused
Higher CPU usage is normal during active detection.
Q: Can I export detection results?
A: Currently, the app displays results in real-time but doesn't export data. You can take screenshots to save results. Export functionality may be added in future versions.

üêõ Troubleshooting

Camera Permission Issues

Problem: Camera not working or "Permission Denied" error

Solution:
  1. Go to iOS Settings ‚Üí Privacy & Security ‚Üí Camera
  2. Find "Object Detector Pro Max" in the list
  3. Toggle the switch to ON
  4. Restart the app

Photo Library Permission Issues

Problem: Can't select photos from library

Solution:
  1. Go to iOS Settings ‚Üí Privacy & Security ‚Üí Photos
  2. Find "Object Detector Pro Max" in the list
  3. Select "All Photos" or "Selected Photos"
  4. Restart the app

App Crashes or Freezes

Problem: App crashes or becomes unresponsive

Solution:
  1. Force close the app (swipe up from app switcher)
  2. Restart your iPhone
  3. Ensure iOS is updated to latest version
  4. If problem persists, try reinstalling the app
  5. Contact support if issue continues

Poor Detection Accuracy

Problem: Objects not detected or too many false positives

Solution:
  1. Adjust confidence threshold in Settings
  2. Improve lighting conditions
  3. Reduce scene clutter
  4. Ensure objects are clearly visible
  5. Try different detection modes
